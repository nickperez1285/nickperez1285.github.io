Ruby "Zip" Method

A method in Ruby is defined as a"... bundle [of] one or more repeatable statements into a single unit.". One particular method that I have found very useful is the ".zip" method. This method will take an single array and merge it's elements with one more other arrays creating one single array consisting of sub-arrays with corresponding elements. The following is a basic example of what the .zip method will return when dealing with two simple arrays:

array1 = [a,b,c]
array2 = [1,2,3]
